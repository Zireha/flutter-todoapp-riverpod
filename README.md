# TodoApp with Riverpod


This project is just the same as the to-do app I made previously. Only the difference is in the state management.

## 📋 Project Scope
What I do in this project is I remade a to-do app from the [previous version](https://github.com/Zireha/flutter-todoapp), which was made with Provider

## 📖 What I learned
From this project, I learned how to use Riverpod as state management, as Riverpod is one of the many state management commonly used in Flutter

## 🔎 Features
The feature of this app is just the same as a common to-do app, which is to make tasks and label them with colored priorities, delete tasks, and edit tasks
